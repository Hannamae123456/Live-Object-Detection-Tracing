🎥 Live Object Detection & Tracing App

A real-time object detection and tracking web application built using Streamlit, YOLOv8, and WebRTC.
This project detects, identifies, and tracks objects live through your webcam directly in the browser.

🚀 Features
📷 Live webcam streaming
🤖 Real-time object detection using YOLOv8
🔍 Object tracking with persistent IDs
⚡ Fast and lightweight performance
🌐 Browser-based interface with Streamlit
🧠 AI-powered computer vision
🎯 Smooth frame processing using WebRTC
🛠️ Technologies Used
Python
Streamlit
YOLOv8
OpenCV
streamlit-webrtc
PyAV
Torch
